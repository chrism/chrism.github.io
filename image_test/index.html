Testing images for use on Orange Radio grid

The aim of this test is to see if any image format renders the assets correctly with no scaling / distortion — it tests whether or not it is due to the scaling or the use of borders to introduce scaling.

1 — Using 256px (API large) images scaled down to 120px with no border

2 — Using 256px (API large) images scaled down to 118px with 1px grey border

3 — Using 128px (API medium) images scaled down to 120px with no border

4 — Using 128px (API medium) images scaled down to 118px with 1px grey border

5 — Using 120px (created for test) with no border

6 — Using 118px (created for test) with 1px grey border
